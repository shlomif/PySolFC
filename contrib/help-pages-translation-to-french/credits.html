PySol utilise les technologies OpenSource suivantes

  • Le langage de programmation Python
  • The Tcl/Tk GUI Toolkit
  • The SDL Simple DirectMedia Layer
  • The SDL MPEG Player Library
  • The MikMod Sound Library

PySol a été développé à l'aide des technologies OpenSource suivantes

  • Le système d'exploitation Linux
  • The XFree86 X Window System
  • L'environnement de bureau KDE
  • L'éditeur de texte FTE
  • The Inno Setup Win32 Installer
