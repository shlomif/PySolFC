Comment jouer à PySol

Utilisation de la souris

Bouton Gauche:

  • Déplacer les cartes
  • Cliquez sur le talon pour distribuer de nouvelles cartes.

Bouton Droit (ou double-clic Gauche)

  • Déplacer les cartes vers les fondations
  • Jeu rapide (si activé)

Bouton du Milieu (ou Ctrl+clic Droit):

  • Voir les cartes partiellement cachées

Ctrl+Clic Gauche:

  • Affiche en surbrillance toutes les cartes correspondantes.

Maj+Clic Gauche:

  • Affiche en surbrillance toutes les cartes de même valeur.

Jeu à deux mains

Placer trois doigts sur 'A' (déplacer auto), 'S' (annuler) et 'D' (distribuer). Vous pouvez ainsi atteindre 'R' (répéter).

Les gauchers préféreront 'L' (déplacer auto), 'K' (annuler) et 'J' (distribuer).

Jeu automatique

Notez que le jeu automatique peut gâcher le jeu. Evitez de l'activer. Ou n'utilisez que le retournement automatique. Certains jeu désactivent automatiquement cette fonctionnalité, sinon ils n'ont plus aucun intérêt.

Retourner automatiquement

  • Retourne automatiquement toutes les cartes.
Déplacement automatiquement
  • Déplace automatiquement les cartes vers les fondations.
Distribuer automatiquement
  • Distribue automatiquement les cartes à la défausse si elle est vide.
Jeu rapide
  • Utilisez le bouton droit de la souris pour déplacer rapidement les piles de cartes. Ce système n'est pas très intuitif, il n'utilise pas le système d'indice.

L'animation est trop lente...

Malheureusement, il manque un concept de sprite à Tcl/Tk toolkit, ce qui provoque (double-buffer invisible), beaucoup de répétition de dessin lors du déplacement des cartes.

Désactiver l'ombre des cartes, l'ombrage des mouvements possibles, la mosaïque d'arrière-plan de table et les sons améliorera l'affichage.

Les arrière-plans de table sont de mauvaise qualité

Ils ne doivent être activés qu'en mode couleurs vraies - sinon la qualité sera mauvaise en raison du tramage.

Vous pouvez ajouter vos propres mosaïques d'arrière-plan. Pour ce faire, copiez les images dans le répertoire data/tiles ou dans ~/.PySolFC/tiles de votre répertoire personnel.

Notes à propos du système de score

  • La prise en compte des scores commence après le premier mouvement. Même si vous annulez tous les mouvements, les scores ne sera pas annulé.
  • Vous perdrez la partie si vous utilisez un indice ou si vous démarrer la démo.
  • Vous pouvez recommencer à tout moment pour avoir une chance supplémentaire de gagner.
  • Si vous ne voulez pas enregistrer le score d'un jeu perdu, vous pouvez temporairement modifiez les paramètres du joueur.
  • Les parties chargées ne comptent pas.
  • Si vous gagnez une partie sans utiliser Annuler, Jeu rapide ni aucune des autres fonctions d'assistance, vous recevrez une récompense spéciale.
  • Enfin, gardez toujours à l'esprit que ceci est un jeu de Patience. Détendez-vos et amusez-vous.

Raccourcis clavier non-documentés

  • Space - Distribuer
  • S - Annuler
  • Retour arrière - Annuler
  • Ctrl-A - Déplacement + Retournement automatique
  • Ctrl-B - Changer l'arrière-plan des cartes
  • Ctrl-H - Afficher le classement interne en donnant un indice
  • Ctrl-I - Changer de mosaïque de table
  • Ctrl-N - Débuter une nouvelle partie avec le numéro de jeu suivant
  • Ctrl-P - Modifier le nom du joueur
  • Ctrl-U - Jouer la musique suivante
