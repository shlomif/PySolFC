Introduction

Vous vous demandez peut-être "Pourquoi encore un autre jeu de solitaire". La réponse est simple...

Avantages de PySol

  • il contient actuellement plus de 1000 jeux de solitaire différents
  • il est basé sur moteur de solitaire extensible
  • beaucoup de jeux classiques comme le Forty Thieves, le FreeCell, le Klondike et le Spider
  • des jeux spéciaux tels que le Ganjifa, le Hanafuda, le Poker ainsi que des jeux de type Tarot
  • une interface moderne
  • de multiples style de cartes et arrière-plans
  • des mosaïques d'arrière-plan de table
  • annuler et répéter illimités
  • des marques-pages persistant
  • chargement et sauvegarde des jeux
  • des statistiques pour le joueur
  • un système d'indices pour se débloquer
  • des jeux en mode démo
  • un assistant de création de jeux de solitaire
  • il supporte les plug-ins écrits par les utilisateurs
  • des échantillons de sons et des musiques de fond
  • un navigateur d'aide HTML intégré
  • beaucoup de documentation
  • il est compatible avec tous les systèmes Unix/X11, Windows 95/98/2000/XP et Mac OSX
  • il est écrit en 100% pur Python
  • il est distribué sous licence GNU General Public Licence
  • il s'agit d'un Freeware de qualité commerciale
