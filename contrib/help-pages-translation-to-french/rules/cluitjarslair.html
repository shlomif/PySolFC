Cluitjar's Lair

Type Klondike. 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Klondike avec des variantes du Hex A Deck.

Règles

Ce jeu se joue comme le Klondike. Les combinaisons sont formées par ordre décroissant en alternant les couleurs. Chaque carte ou suite de cartes peut être déposée sur un emplacement vide. Les magiciens se jouent selon leur propre valeur dans le tableau, sans tenir compte des couleurs. Si plusieurs magiciens sont posés l'un sur l'autre, les couleurs doivent obligatoirement être alternées pour pouvoir déplacer le groupe de carte entier. Les cartes du talon sont retournées une par une. Il n'y a qu'une seule donne. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Les magiciens ne peuvent être placés sur leur fondation qu'une fois toutes les autres cartes placées sur leurs fondations respectives. De ce fait, la fin de jeu peut devenir un petit casse-tête. Utilisez un emplacement vide pour écarter les magiciens.
