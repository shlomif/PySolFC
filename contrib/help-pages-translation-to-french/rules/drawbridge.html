Drawbridge

Type Klondike. 1 jeu. 2 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Klondike avec des variantes du Hex A Deck.

Règles

Ce jeu se joue comme le Klondike. Les combinaisons sont formées par ordre décroissant en alternant les couleurs. Chaque carte ou suite de cartes peut être déposée sur un emplacement vide. Les magiciens se jouent dans l'ordre de leur valeur dans le tableau, alternant rouge et noir. Les cartes du talon sont retournées une par une. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Réutilisez les cartes déjà placées dans les fondations pour retourner plus facilement les cartes du tableau.
