Five Aces

Type Tarot 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Il s'agit d'un jeu de type Baker's Dozen, joué avec les 78 cartes du jeu de tarot. Les suites sont formées par ordre décroissant en alternant les couleurs. Les Atouts peuvent remplacer n'importe quelle couleur. Les As sont distribués aux fondations en début de partie.

Règles

Les suites se construisent par ordre décroissant en alternant les couleurs, en utilisant les Atouts sur n'importe quelle couleur. Un groupe de cartes peut être déplacé vers une autre colonne, mais un emplacement vide ne peut recevoir qu'une seule carte. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Utilisez les Atouts sur les emplacements vides pour pouvoir recevoir des groupes de cartes.

Auteur

Ce jeu et sa documentation originale sont écrits par T. Kirk.
