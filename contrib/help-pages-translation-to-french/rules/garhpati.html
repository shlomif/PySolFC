Garhpati

Type Mughal Ganjifa 1 jeu. Donnes illimitées.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Ce jeu est similaire au Klondike. Les combinaisons sont formées par ordre décroissant en alternant les couleurs.

Règles

Les sont regroupées par ordre décroissant en alternant les couleurs. Les fondations se construisent par ordre croissant et par couleur de cartes, en commençant par l'As. Seuls les Mirs (rois), peuvent être déposés sur un emplacement vide. Le jeu de Mughal Ganjifa comporte 8 couleurs. Le fait de devoir trouver quelles couleurs correspondent sur le tableau ajoute un peu de piment à ce jeu. Si une carte ne peut pas se placer dans une suite, essayez-en une autre de la même valeur. Les cartes du talon sont retournées par trois. Les donnes sont illimitées. Les cartes des fondations ne peuvent être replacées sur le tableau.

Ce jeu fait partie d'une série de jeux dont les noms finissent par "pati", ce qui signifie "seigneur de". Garhpati signifie "Seigneur des forts". Les noms sont les noms des couleurs de cartes d'un jeu de Ganjifa à douze couleurs.

Stratégie

Essayez de dégager le tas de cartes le plus bas d'abord. Déplacer les cartes du tableau pour pouvoir jouer celles de la réserve.
