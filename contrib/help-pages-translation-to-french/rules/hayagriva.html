Hayagriva

Type Dashavatara Ganjifa 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Les suites sont construites par ordre décroissant, uniquement sur le tableau, dans la limite de 12 par colonne. Seuls les Mirs peuvent être déposés sur un emplacement vide.

Règles

Toutes les cartes sont distribuées aux seize colonnes en début de partie. Les suites sont construites par ordre décroissant. Les fondations sont construites par ordre croissant et par couleur. Seul un Mir ou une suite peuvent être déposés sur un emplacement vide. Les quatre réserves peuvent recevoir une carte chacune.

Stratégie

Un emplacement libre peut s'avérer plus utile que les réserves. Essayez de libérer des emplacements.
