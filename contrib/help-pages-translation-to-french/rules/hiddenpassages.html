Hidden Passages

Type Klondike. 1 jeu. 2 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire à Blind Alleys, avec des variantes de Hex A Deck.

Règles

Le jeu se joue comme Blind Alleys. Les combinaisons sont formées par ordre décroissant en alternant les couleurs. Chaque carte ou suite de cartes peut être déposée sur un emplacement vide. Les magiciens se jouent dans l'ordre de leur valeur dans le tableau, alternant rouge et noir. Les cartes du talon sont retournées une par une. Les As sont placés sur les fondations en début de partie. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Les magiciens ne peuvent être placés sur leur fondation qu'une fois toutes les autres cartes placées sur leurs fondations respectives. De ce fait, la fin de jeu peut devenir un petit casse-tête.
