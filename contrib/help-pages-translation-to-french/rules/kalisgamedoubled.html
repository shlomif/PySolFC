Double Kali's Game

Type Dashavatara Ganjifa. 2 jeux. 4 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire à Lara's Game, avec 12 tas de cartes, quatre donnes et une réserve.

Règles

Référez-vous à la description concernant la distribution des cartes du jeu Lara's Game. Les différences consistent à ce que les cartes sont distribuées sur 13 tas au lieu de 14 et si une carte distribuée est un Wazir ou plus élevée, une carte sera déposée sur le talon. Ce jeu utilise un jeu de 120 cartes de Dashavatara. Pour le reste, les règles de distribution sont les même.

Le jeu se joue de ma même manière que le jeu Lara's Game, excepté qu'il y a 4 donnes. Quand le talon est vide, après chaque donne, les cartes sont ramassées sur le tableau et redistribuées sans être mélangées, en utilisant les mêmes règles que pour la donne initiale. Les fondations reçoivent chacune deux séries de cartes. Les fondations sont construites de façon circulaire. Un As est placé sur un Mir.

La réserve peut recevoir quatre cartes du tableau.

Stratégie

Utilisez la réserve pour vous aider à dégager les cartes du dessous. Le meilleur moment pour l'utiliser est juste avant une donne. Lorsqu'une carte est placée sur la réserve, elle ne peut-être déplacée que vers une fondation.
