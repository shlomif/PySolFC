Relaxed Katrina's Game

Type Tarot 2 jeux. Deux donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire à Lara's Game, avec 22 tas de cartes et deux donnes, en utilisant un jeu de tarot. Il s'agit du jeu Katrina's Game, avec une réserve.

Règles

Référez-vous à la description concernant la distribution des cartes du jeu Lara's Game. Les différences consistent à ce que les cartes sont distribuées sur 23 tas au lieu de 14 et si une carte distribuée est un Valet ou plus élevée, une carte sera déposée sur le talon. Pour le reste, les règles de distribution sont les même.

Le jeu se joue de ma même manière que le jeu Lara's Game, à deux exceptions près. La première est que vous avez droit à deux donnes. Quand le talon est vide après la première donne, les cartes sont ramassées sur le tableau et redistribuées sans être mélangées.

La seconde est la présence d'une réserve à droite des fondations. Cette réserve accepte deux cartes. Les cartes de la réserve ne peuvent être placées que sur les fondations.

Stratégie

Si deux cartes identiques peuvent être déplacées vers les fondations, prenez celle se trouvant sur le tas contenant le plus de cartes. Déplacer les cartes d'un ensemble de fondations à l'autre vous permettra de prolonger le jeu.
