King Only Hex A Klon

Type Klondike. 1 jeu. 2 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Klondike avec des variantes du Hex A Deck.

Règles

Le jeu se joue comme le Klondike avec les magiciens comme jokers. Ils peuvent être placés sur le tableau pour remplacer une carte de n'importe quelle valeur ou couleur. Quand un magicien est placé dans une suite, celle-ci ne peut plus être déplacée. Si un magicien remplace une carte de sa propre valeur dans une suite, elle peut être déplacée. Une suite contenant deux magiciens ne peut être déplacée que s'ils remplacent une carte de leur propre valeur, et ne sont pas adjacents. Les magiciens ne peuvent être retirés du tableau que quand toutes les autres cartes ont rejoint les fondations. Seuls les Rois peuvent être placés sur un emplacement vide. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Essayez de ne pas poser de magicien dans les colonnes dans lesquelles il y a encore des cartes retournées. Quand toutes les cartes sont ouvertes, essayez de mettre les magiciens de côté aussi vite que possible. Placer les tous sur un emplacement vide. Un magicien est distribué au bas de deuxième colonne à partir de la droite. Si vous déplacez ce magicien alors qu'il n'y en a pas d'autre sur le tableau, vous ne pourrez pas gagner le jeu.
