Klondike Plus 16

Type Klondike. 1 jeu. 2 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Klondike avec des variantes du Hex A Deck.

Règles

Le jeu se joue comme le Klondike avec les magiciens comme jokers. Ils peuvent être placés sur le tableau pour remplacer une carte de n'importe quelle valeur ou couleur. Quand un magicien est placé dans une suite, celle-ci ne peut plus être déplacée. Les magiciens on un ordre de valeur, comme les couleurs de cartes, correspondant à une suite allant de l'As au 4. Si un magicien remplace une carte de sa propre valeur dans une suite, elle peut être déplacée. Sa valeur est déterminée par certains éléments de comparaison. Le magicien correspondant à l'As a le chapeau le plus grand. Une suite contenant deux magiciens ne peut être déplacée que s'ils remplacent une carte de leur propre valeur, et ne sont pas adjacents. Les cartes du talon sont retournées une par une. Seuls les Rois, peuvent être déposés sur un emplacement vide. Les magiciens peuvent être déplacés vers leur fondation à n'importe quel moment et dans l'ordre de leur valeur. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Les magiciens peuvent être déplacés vers leur fondation à n'importe quel moment, mais peuvent s'avérer utile pour pouvoir déplacer ds cartes sur le tableau.
