Kurma

Type Dashavatara Ganjifa 1 jeu. Donnes illimitées.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Ce jeu est similaire au Klondike. Les suites sont construites par ordre décroissant et par enseigne.

Règles

Sur le tableau, les cartes sont rangées par ordre décroissant de valeur, dans la même enseigne. Les fondations se construisent par ordre croissant et par couleur de cartes, en commençant par l'As. Les cartes du talon sont retournées une par une. N'importe quel carte ou suite de cartes peut être déposée sur un emplacement vide. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Les probabilités de victoire sont assez faibles. Déplacez les cartes des piles les plus basses d'abord.
