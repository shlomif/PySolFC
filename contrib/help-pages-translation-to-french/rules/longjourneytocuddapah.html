Long Journey to Cuddapah

Type Braid. Deux jeux. Trois donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Braid joué avec deux jeux de Dashavatara Ganjifa.

Règles

Le jeu se joue comme le Braid. Dans cette variante, vous avez deux cordes, dont chacune a sa corde de réserve. Les fondations sont au nombre de 20, placées aux deux extrémités. Les deux colonnes suivantes sont les dix cordes de réserve. Il y a également deux colonnes de cinq réserves chacune. Les deux colonnes intérieures sont les deux cordes. Chaque corde reçoit 20 cartes. Lorsqu'une corde de réserve se libère, la carte en haut de la corde correspondante y est déposée. Lorsque toutes les cartes d'une corde sont retirées, une carte de l'autre corde est utilisée.

Cuddapah est une cité indoue ayant une histoire concernant la production de cartes de Ganjifa.

Stratégie

Créez des suites où vous pouvez, après avoir tiré une carte correcte du talon. Ce type de jeu est très stratégique et demande beaucoup d'attention.
