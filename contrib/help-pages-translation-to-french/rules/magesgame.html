Mage's Game

Type Baker's Dozen 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au jeu Baker's Dozen, avec 12 colonnes de cartes alternant les couleurs et des variantes du Hex A Deck.

Règles

Le jeu se joue comme le jeu Baker's Dozen. Les combinaisons sont formées par ordre décroissant en alternant les couleurs. Les magiciens se jouent dans l'ordre de leur valeur dans le tableau, alternant rouge et noir. Chaque carte ou suite de cartes peut être déposée sur un emplacement vide. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Essayez de libérer des emplacements.
