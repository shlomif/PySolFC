Matrix

Type Matrice. 1 jeu. 1 donne.

But du jeu

Replacez les pièces du jeu dans leur ordre respectif.

Règles

Au début de la partie, toutes les pièces excepté la pièce finale sont place dans l'ordre sur le tableau. Les pièces sont ensuite mélangées. Avec des grilles plus grandes, cela peut prendre quelques secondes. Les pièces ne sont pas déplacées à travers le jeu. Elles sont déplacées en cliquant une pièce située sur la ligne ou la colonne ou se situe l'emplacement vide. La pièce et toutes celles autour se déplaceront d'un cran vers l'emplacement vide. Il est conseillé d'utiliser les styles de pièces prévus pour les dimensions de la grilles utilisée. Les dimensions sont indiquées dans le nom du style de pièces. King of Hearts 4x4, Players Trumps 10x10, etc. Les styles par défaut fonctionnement avec toutes les dimensions de grilles. Quand toutes les pièces ont été placées dans le bon ordre, la pièce finale est placée sur le tableau et la partie est gagnée. Tous les jeux de type Matrice peuvent être résolus.

Stratégie

Commencez dans le coin supérieur gauche et finissez une rangée avant d'en commencer une autre. Prenez une capture d'écran des images avant que le jeu ne soit mélangé.
