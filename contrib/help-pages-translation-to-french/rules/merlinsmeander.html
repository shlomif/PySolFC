Merlin's Meander

Type Braid. Deux jeux. Trois donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Similaire au Klondike avec des variantes du Hex A Deck.

Règles

Le jeu se joue comme le Braid. Dans la variante Hex A Deck, les Magiciens n'ont pas de fondations propres. Il y a deux réserves à gauche des fondations, pouvant recevoir jusqu'à 3 Magiciens chacune. La partie est gagnée lorsque toutes les cartes sont placées sur les fondations.

Stratégie

Créez des suites où vous pouvez, après avoir tiré une carte correcte du talon. Laissez les emplacements réservés aux Magiciens libres tant que vous n'en avez pas besoin.
