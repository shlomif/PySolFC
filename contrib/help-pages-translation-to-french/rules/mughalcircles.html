Mughal Circles

Type Mughal Ganjifa 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Les suites se construisent par ordre décroissant et par couleur de cartes. N'importe quelle carte peut être placée sur un emplacement libre. Une seule carte peut être déplacée à la fois.

Règles

Toutes les cartes sont distribuées aux 24 colonnes en début de partie. Sur le tableau, les cartes sont rangées par ordre décroissant de valeur, dans la même enseigne. Les fondations sont construites par ordre croissant et par couleur. N'importe quelle carte peut être placée sur un emplacement libre. Les deux réserves peuvent recevoir une carte chacune. Une seule carte peut être déplacée à la fois.

Stratégie

Essayez de garder toujours une réserve disponible. Placez les cartes les plus fortes sur les emplacements vides.
