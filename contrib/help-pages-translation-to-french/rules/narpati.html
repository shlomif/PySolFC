Narpati

Type Mughal Ganjifa 1 jeu. 1 donne.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Ce jeu est similaire au Klondike. Les combinaisons sont formées par ordre décroissant en alternant les couleurs.

Règles

Les sont regroupées par ordre décroissant en alternant les couleurs. Les fondations se construisent par ordre croissant et par couleur de cartes, en commençant par l'As. Seuls les Mirs (rois), peuvent être déposés sur un emplacement vide. Le jeu de Mughal Ganjifa comporte 8 couleurs. Le fait de devoir trouver quelles couleurs correspondent sur le tableau ajoute un peu de piment à ce jeu. Si une carte ne peut pas se placer dans une suite, essayez-en une autre de la même valeur. Les cartes du talon sont retournées par trois. Il n'y a qu'une seule donne. Les cartes des fondations peuvent être replacées sur le tableau.

Ce jeu fait partie d'une série de jeux dont les noms finissent par "pati", ce qui signifie "seigneur de". Narpati signifie "Seigneur des Hommes". Les noms sont les noms des couleurs de cartes d'un jeu de Ganjifa à douze couleurs.

Stratégie

Essayez de dégager le tas de cartes le plus bas d'abord. Déplacez les cartes sur le tableau pour libérer des emplacements pour les cartes de la défausse.
