Two Familiars

Type Klondike. Deux jeux. 2 donnes.

But du jeu

Déplacez toutes les cartes vers les fondations.

Description rapide

Il s'agit d'une version à 2 jeux de cartes de The Familiar.

Règles

Ce jeu se joue comme le Klondike. Les combinaisons sont formées par ordre décroissant en alternant les couleurs. Les Magiciens prennent la couleur rouge ou noire. Seuls les 10 (cartes les plus fortes) peuvent être placés sur un emplacement vide. Il y une réserve peut recevoir jusqu'à 3 Magiciens. Aucune autre carte ne peut y être déposée. Les Magiciens pourront être placés sur le tableau dans l'ordre de leur valeur. Ils peuvent remplacer n'importe quelle carte, rouge ou noire. Les cartes du talon sont retournées une par une. Les cartes des fondations peuvent être replacées sur le tableau.

Stratégie

Attention à la manière dont vous placez les Magiciens sur la réserve. Si vous placez un Magicien de valeur inférieure sur un autre de valeur supérieure, il vous faudra l'enlever avant d'envoyer celui de valeur supérieure vers la fondation.
