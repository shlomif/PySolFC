Fortune's Favor

De Wikipédia, l'encyclopédie libre

Fortune's Favor est un jeu de cartes de solitaire utilisant 1 jeu de 52 cartes. On lui a probablement donné ce nom, car le joueur a toutes les chances de gagner.

Tout d'abord les 4 As sont retirés du jeu et placé sur une rangée pour former les bases des fondations. Ces fondations sont construites par ordre croissant par couleur de cartes jusqu'aux Rois.

Sous les fondations, 2 rangées de 6 cartes sont distribuées (on peut préférer une disposition de 12 cartes). Elles forment la base des 12 piles du tableau. Les cartes du dessus du tableau peuvent être placées sur les fondations ou dans le tableau. Dans le tableau, les suites sont construites par ordre décroissant par couleur de cartes. Les espaces vides seront réapprovisionnés par la défausse ou, s'il n'y en a pas, par le talon. Une seule carte peut être déplacée à la fois.

Quand le jeu bloque, une carte du talon est placée sur la défausse. La cartes de dessus de celle-ci peut être jouée sur le tableau ou sur les fondations.

La partie est gagnée lorsque toutes les cartes sont placées sur les fondations.
