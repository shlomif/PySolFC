Gate

De Wikipédia, l'encyclopédie libre

Gate est un jeu de cartes de solitaire utilisant 1 jeu de 52 cartes. Il doit son nom au fait que les cartes sont placées de façon à former un portail.

Tout d'abord, on place 2 colonnes de 5 cartes, face visible. C'est la réserve ou les "piliers du portail". Ensuite, entres ces colonnes, on place 2 rangées de 4 cartes, face visible. Ce sont les "barreaux" ou le tableau. Les fondations viennent au-dessus de la première rangée de cartes.

Le but du jeu, comme touts les jeux de solitaire, est de trouver les As, de les placer sur les fondations et de construire des suites croissantes par couleur de cartes jusqu'aux Rois.

Les cartes des barreaux peuvent être placées sur les fondations ou sur les autres cartes des barreaux. Sur les barreaux, les suites se construisent par ordre décroissant en alternant les couleurs (une carte rouge sur une noire et vice-versa). Les espaces entre les barreaux sont réapprovisionnés par les cartes des piliers. Quand toutes les cartes des piliers sont utilisées, la première carte de la défausse ou du talon, si la défausse est vide peut être placée sur les espaces vides. Les piliers du portail ne sont jamais réapprovisionnés.

Généralement, une seule carte peut être déplacée à la fois. La règle prédominante concernant les déplacements autorise à déplacer des suites entières. Il existe cependant une règle qui interdit le déplacement de groupes de cartes, ce qui rend le jeu plus difficile (mentionnée dans Card Games Made Easy par Marks and Harrod, ISBN 1-899606-17-3).

Les cartes du talon sont distribuées une par une à la défausse. La première carte de celle-ci peut être placée sur les fondations ou sur les barreaux, ou sur un espace vide entre les barreaux. Il n'y a qu'une seule donne.

Le jeu s'achève une fois le talon épuisé. La partie est gagnée lorsque toutes les cartes sont placées sur les fondations.
