Deck

One-Deck game type. 1 deck. No redeal.

Object

Discard all but the last four cards.

Rules

Four cards are dealt to four tableau piles. If two of these cards are of the same suit, they can be discarded. At any point, four additional cards can be dealt from the stock to these four piles.

The game is won if all cards are discarded except for the last card of each suit.

Notes

Deck is a variation of Cover. It is also related to Aces Up.

Autodrop is disabled for this game.
